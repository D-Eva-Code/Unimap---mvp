# UniMap+

UniMap+ is a campus-focused platform built for **UNIBEN (University of Benin)**. It allows **students, vendors, and drivers** to interact seamlessly: students can place orders, vendors can manage menus and orders, and drivers can fulfill deliveries. In addition to food delivery, UniMap+ **helps new students and visitors navigate around campus**, making campus life easier and more organized.

## Features

- Role-based authentication: Student, Vendor, Driver  
- Secure login using **JWT**  
- Student: browse vendors and menus, place orders  
- Vendor: manage menu, view incoming orders  
- Driver: view orders for delivery  
- **Campus navigation:** helps new students and visitors find their way around UNIBEN  
- Checkout with integration-ready structure (e.g., Paystack)  
- Responsive and user-friendly frontend  

---

## Tech Stack

- **Frontend:** React (Vite), React Router DOM, CSS  
- **Backend:** Node.js, Express.js  
- **Database:** MySQL  
- **Authentication:** JWT, bcrypt for password hashing  
- **Other:** dotenv, cors
